# Contributing to Freedom Loader
Welcome! Thanks for being interested in Freedom Loader. Your contributions-bug reports, feature ideas, or code improvements-are always appreciated.

## How to Contribute
1. **Report an issue**  
   Found a bug, weird behavior, or have an idea? Open an issue. The clearer it is, the faster we can fix it. <br>

2. **Submit a pull request**  
   To fix a bug, add a feature, or improve docs:
   - Fork the repository  
   - Create a branch: <span class="code-span"> git checkout -b feature/my-awesome-feature</span> 
   - Make your changes  
   - Add tests if needed  
   - Make sure everything works  
   - Open a PR to the <span class="code-span">main</span> branch <br>

3. **Improve the documentation**  
   Docs are important: README, wiki, guidesâ€¦ If something is unclear, improve it!

## Guidelines
- Keep PR titles and descriptions clear and concise.  
- Document what you change and why.  
- Follow the existing code style.  
- If adding features, include tests or verification steps.

## Review & Merge
- After submitting a PR, it will be reviewed. We may ask for changes or suggest improvements.  
- Thanks for your patience: we try to respond quickly but sometimes have other priorities.

## Code of Conduct
Please contribute respectfully and constructively. Everyone is welcome as long as discussions remain polite and productive.
