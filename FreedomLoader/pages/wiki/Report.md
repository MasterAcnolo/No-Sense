# How to Report Issues in Freedom Loader
Thanks for taking the time to report a problem! The more info you provide, the faster we can reproduce and fix it.

## What to Include
When creating an issue, include:


1. **Clear, descriptive title**  
   Example: *‚ÄúCrash when trying to download [10 Hours of Shrimpt Quartet](https://www.youtube.com/watch?v=9btGoo-F5FY)‚Äù*


2. **Description of the problem**

3. **Context**  <br>
   - Freedom Loader version (Written in the window title, or in the logs) <br>
   - Operating system (Windows version, etc.)  <br>
   - Any other relevant info (antivirus, admin rights, etc.) <br>

      <span class="code-span">C:\Users\[USERNAME]\AppData\Local\FreedomLoader\logs\LOGS-20xx-xx-xx.log</span>


4. **Steps to reproduce**  
   Clear instructions help us reproduce the bug quickly.


5. **Workaround (if you know one)**  
   If you found a temporary fix, include it.
